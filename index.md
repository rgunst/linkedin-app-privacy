Purpose of the Application
This application uses LinkedIn’s API for authentication and limited profile access for development and experimentation purposes.

Data Collected
The application may access the following information via LinkedIn OAuth:

The data accessed is used solely for authentication and testing purposes. No personal data is sold, shared, or distributed to third parties.

No personal data is stored permanently unless explicitly required for application functionality. Any stored data is protected using industry-standard security practices.
